Getting Started with Connectify

  1. Home page par Sign Up button click karein
  2. Apna email ya phone number enter karein
  3. Full name, username aur password set karein
  4. Verification code enter karein (email/SMS par aayega)
  5. Profile picture aur bio add kar sakte hain (optional)

Login karne ke do tarike hain:

  • Username/Email + Password: Apna registered username/email aur password daalein
  • Mobile Number: Apna mobile number daalein aur OTP se login karein

Agar password bhool gaye to "Forgot Password" click karein

Creating Content

  1. Plus Button (+) par click karein jo left sidebar ya mobile bottom navigation me hai
  2. Post option select karein
  3. Photo/video upload karein (multiple select kar sakte hain)
  4. Caption, location aur tags add karein
  5. Advanced settings adjust karein (hide likes, disable comments)
  6. Share button click karein

Community & Interaction

Follow / Unfollow kaise karein?

Kisi user ko follow karne ke liye:

  • Profile par jayein aur Follow button click karein
  • Ya search results me Follow button use karein
Like kaise karein?

Post ke neeche heart icon ❤️ par click karein. Double-tap bhi kar sakte hain image par.

Comment kaise karein?

Post ke neeche comment box me likhein aur Enter press karein.

Share kaise karein?

Paper plane icon ✈️ par click karein. Direct message ya copy link kar sakte hain.

Save kaise karein?

Post ke right side bookmark icon 🔖 par click karein. Saved posts profile me dekhein.

Account & Profile Settings

  1. Apne profile par jayein
  2. Edit Profile button click karein
  3. Profile picture, name, username, bio, website change karein
  4. Save changes karein

Settings → Change Password me jayein. Old aur new password daal kar update karein.

Privacy & Security

Account private kaise karein?

Settings → Privacy → Private Account toggle on karein. Sirf approved followers hi aapki posts dekh payenge.

Kisi ko block kaise karein?

Unke profile par jayein, teen dots menu se Block select karein.

Report kaise karein?

Post ya profile ke teen dots menu se Report option use karein.

Apna data download kaise karein?

Settings → Privacy → Download Data se request karein. 48 hours me email par link aayega.

Troubleshooting

Follow / Like / Comment working nahi?
  • Page refresh karein (Ctrl+F5 ya Cmd+Shift+R)
  • Browser cache clear karein
  • Kisi aur browser ya incognito mode me try karein
  • Internet connection check karein
  • Kuch minutes baad try karein (server busy ho sakta hai)
Photos upload nahi ho rahi?
  • File size check karein (max 10MB recommended)
  • Format check karein (JPG, PNG, GIF, MP4 supported)
  • Network speed test karein
Notifications nahi aa rahe?
  • Browser notifications enabled hain ya nahi check karein
  • Settings me notification preferences check karein
  • Phone/browser do not bat mode off karein
App slow chal raha hai?
  • Browser cache clear karein
  • Unnecessary tabs close karein
  • Internet speed check karein
  • System restart karein
